## Getting Started

### Prerequisites

* yarn
  ```sh
  npm install --global yarn
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/QuickSoOrt/file-management-system-backend.git
   ```
2. Install NPM packages
   ```sh
   yarn install
   ```
3. Enter your API in `config.js`
   ```js
   const API_KEY = 'ENTER YOUR API';
   ```

8. Run the project
   ```sh
   yarn start:dev
   ```
